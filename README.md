##  Project Overview

**Business Context**  
Smatfin aims to formally launch its online store in Zambia. Following the store's integration with the website in January 2023, promotion through social media and paid ads has yielded low conversion rates, largely due to local preference for WhatsApp-based shopping. This analysis leverages Google Analytics data to bridge the gap between current performance and market realities.

**Objective**  
To analyze website traffic, user engagement, and audience characteristics, translating findings into actionable recommendations for optimizing the online store's performance and aligning with user preferences.

##  Key Analysis Areas

The analysis addresses the following core business questions:

| Analysis Area | Business Question |
|--------------|-------------------|
| **Channel Performance** | Which marketing channels drive the most traffic? |
| **User Growth Trends** | How are traffic sources contributing to user acquisition over time? |
| **Geographic Distribution** | Is the audience from target regions, particularly Zambian commercial hubs? |
| **Device Usage** | What devices do visitors use, and what are the UX implications? |
| **User Retention** | How effectively does the website retain visitors? |
| **Audience Segmentation** | *Note:* Demographic and interest data were insufficient for reliable segmentation |

##  Strategic Recommendations

Based on the analysis, the following recommendations support the online store launch:

1. **Geographically-Targeted Campaigns**  
   Deploy targeted advertising in Lusaka and Kitwe, where user potential is highest.

2. **Regional Content & Offers**  
   Develop region-specific content and promotions tailored to local preferences.

3. **Mobile Experience Optimization**  
   Prioritize responsive design, fast load times, and seamless mobile UX, given mobile user dominance.

4. **Source-Specific Landing Pages**  
   Create dedicated landing pages for different traffic sources to enhance relevance and user experience.

5. **Bounce Rate Reduction**  
   Implement UX/UI improvements, enhance content quality, and optimize page speed.

6. **Conversion Path Optimization**  
   Analyze user flow to identify drop-off points and strategically integrate store products within relevant content.

7. **Targeting Strategy Refinement**  
   Continuously assess and adjust marketing channels to attract audiences with higher purchase intent.
